## Context

Michael wants to re-run the CLI-Anything real estate brainstorm, now that we've expanded the compatibility model to include proprietary software (Photoshop, InDesign, AutoCAD, SketchUp, DaVinci Resolve, Excel, etc.). The key insight: different types of RE investors have completely different workflows, software needs, and pain points. A generic brainstorm misses this. We need to segment first, then ideate per segment.

Two-phase approach:
1. Identify RE investor segments and map their specific activities, software, pain points
2. Run creative brainstorming per segment using the full software universe

Previous brainstorm (46 ideas) was generic. This one should produce segment-specific, actionable pipelines.

### Risks
- Too many segments dilutes the quality per segment
- Some segments may overlap significantly
- Proprietary software access claims need to be realistic (not all scripting APIs are equal)
